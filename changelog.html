New Commands

/start - Start a NebCoins account
/balance - View your NebCoins balance
/farm - View your farm
/plant - Plant or view list of plants
/inventory - View your inventory
/sell - Sell an item from your inventory
/market - View today's market
/work - Work for NebCoins
/leaderboard - View the leaderboard
/level - View your NebCoins level
/transfer - Transfer NebCoins
/beg - Beg for NebCoins
/journey - Go on a journey to find pets
/mypets - View your pets
/pets - View a list of pets

Extra Adds

Added a Bug Report System
• Every time an error occurs in the code, it informs the user "an error occured" and asks the user if they want to report it.
• If reported, the error will directly go to the developer and will be fixed as soon as possible
