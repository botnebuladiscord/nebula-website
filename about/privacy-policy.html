Privacy Policy

Nebula abides to the data privacy policy as recommended by discord.

Data Collection

Nebula collects information such as User IDS, Message IDS, Channel IDS. This data is not used for any other purposes except the running of the bot. You can find information on why Nebula uses this data in Section 2. To request a deletion of data, please look at Section 4.
Data is deleted after 30 days in case of inactivity



Reason for data collection

  1. User IDS: Nebula stores user IDS for storing information such as your NebCoins account.
  2. Message IDS: Nebula stores message ids for further interactions such as using the Message ID to run a poll created before (poll data is deleted after poll ends).
  3. Channel IDS: Nebula stores channel ids for going back to a message (Discord.py needs a channel to access a message in it) such as in polls.



Duration of data storage

  1. NebCoins: NebCoins data is deleted after 30 days if you are inactive or it is deleted if you request a data deletion.
  2. Poll Data: Poll data is erased after the poll ends, or if you request deletion of data



Deletion of data

To request a complete data deletion for yourself you can either send an Email at - bot.nebula.discord@gmail.com or Send a modmail in the support server
(https://discord.gg/CxtY6rTnr4)
